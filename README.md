## 夏休み課題(ゲームデザイン実習、ゲームプログラミング共通)

UnityのWebGL、あるいは、clusterで、オリジナルの要素やルール、キャラクターを一つ以上加えた作品を完成させて、前者なら[unityroom](https://unityroom.com/)、後者なら[cluster](https://cluster.mu/)にアップロードする。公開状態は、限定公開や非公開でよい。

以下のような様々な取り組み方がある。

- 「よけとる for Unity」の画面レイアウトを変えたり、操作方法を変えたり、キャラクターの性質を変えて別物にする
- Unityの書籍、チュートリアル、ブログなどから良さそうなものを見つけて、手順通りに完成させたものにアレンジを加える
- アセットストアからよさそうな完成プロジェクトを見つけて改造する
- もちろん、完全オリジナルでも構わない

以下を守ること。

- **プログラムの変更や追加があること**。絵や音、フォントのみの変更は不可
- フリー素材を利用した場合は、その素材の著作者名と入手元URLを記録しておくこと。unityroomには使用アセットの登録欄があるので利用するとよい
- ゲームの概要、操作方法、遊び方を簡潔にまとめておく
- アップロードができない、不具合の原因が分からないなどあったら、できるところまで進めておいて、夏休み明けの初回の講義で解消する。slackでの質問も可

夏休み明けに発表予定。

## 講義準備
　1つのラベルが動いて、跳ね返って、マウスカーソルと重なると停止するところまで完成しているプロジェクトをフォークして、その続きを完成させてください。

- https://github.com/tanakaedu/ArrayPractice を開く
- **Code**ボタンをクリックして、**Download ZIP**を選択
- ダウンロードが完了したら、エクスプローラーを開いて、ダウンロードフォルダーを選択
- ダウンロードした`ArrayPractice.zip`を、ドキュメントの自分のフォルダーへ移動
- 移動した`ArrayPractice.zip`を右クリックして、7-Zip > “ArrayPractice”に展開
- 展開したフォルダーを、いつも通り[GitHubに新しいプロジェクトで登録してPublishする](https://github.com/datgm20/gp1/wiki/VisualStudio%E3%81%AE%E3%83%97%E3%83%AD%E3%82%B8%E3%82%A7%E3%82%AF%E3%83%88%E3%82%92GitHub%E3%81%ABPublish%E3%81%99%E3%82%8B)
- Visual Studio 2019を起動して、**プロジェクトやソリューションを開く**を選択して、展開した`ArrayPractice`フォルダー内の`ArrayPractice.sln`を開く

　以上できたら、以下を実施。

- label1をコピー&ペーストして3つに増やす
- 変数やコードをコピー&ペーストで増やして、以下をできるだけ進める。**成果が出るごとにコミットとプッシュ**をする。すでに配列や繰り返しが使えるようなら使って構わない
  - 3つのラベルを動かす
  - 3つのラベルを個別の速度で動かす
  - 3つのラベルがそれぞれフォームの端で跳ね返るようにする
  - どれか一つでもマウスがラベルに重なったタイマーが止まるようにする
  - マウスが重なった時、タイマーは止めずにラベルが消えるようにする
  - 3つラベルが消えたらタイマーを止めるようにする
  - 時間が余ったら自由に改造する
